Killability - Description
======================
Draws Killmarker/Combo if you can Kill the Champion.
http://i.imgur.com/iGkeohP.png

Supported Spells
======================
Deathfire Grasp (first in combo)  
Tiamat  
Ravenous Hydra  
Hextech Gunblade  
Bilgewater Cutlass  
Blade of the Ruined King  
Ignite  
Q/W/E/R  

Config Options
======================
Display Kill Icon  
Display Combo Text
